BUS PASS MANAGEMENT SYSTEM

The Bus Pass Management System is a web-based application designed to streamline the process of issuing, managing, and tracking bus passes for students and commuters. The system allows users to apply for passes online, check their status, and renew or cancel passes with ease. Administrators can manage user accounts, track pass issuance, and generate reports to analyze usage trends.

FACE MASK DETECTION

The Face Mask Detection System is an intelligent application that uses computer vision to monitor and ensure compliance with mask-wearing protocols in public spaces. By leveraging deep learning algorithms, the system can accurately identify individuals wearing masks and those not wearing masks in real-time. This project aims to enhance safety measures during health crises, such as pandemics, by providing automated monitoring and alerts.
